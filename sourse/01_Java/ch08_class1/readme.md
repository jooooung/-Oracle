# 클래스 - 객체지향 프로그램의 기본 코딩방법
## 1. 클래스 제작 방법
1. 패키지명 
2. 클래스명   (1,2는 클래스 만들면서 설정하면 저절로 작성된다)
3. 데이터(인스턴트 변수=멤버변수, 필드) : 이 데이터는 생성자나 setter를 이용해서  
 초기화하지 않으면 객체는 null, 숫자는 0, boolean은 false로 초기화되어 들어간다
4. 생성자함수 : 클래스명과 똑같이 리턴타입이 없는 메소드를 생성자라 하며 처음 클래스형 객체를 만들때 호출된다.  
 모든 클래스는 반드시 하나 이상의 생성자가 있어야 한다.  
 (만들지 않는다면 자동으로 디폴트 생성자를 만들어준다)
```
1.package com.lec.ex;
  2.public class ExClass { // 반드시 대문자로 시작. 알파벳,숫자,_
	  3.private 자료형 인스턴스변수( = 속성 = 필드)명;
	  4.public ExClass(){ 	}
	  5.public return_type method(){
	  . . .
    }
}
```
## 2. 생성자
- 매개변수 없는 생성자, 매개변수 있는 생성자 등 여러가지 생성자가 있다  
- 1 에서도 말했듯 생성자가 없다면 디폴트 생성자가 자동 생성되고  
생성자가 하나라도 있다면 디폴트 생성자는 생성되지 않는다
